Assignment

Your assignment for this week will be to build an express app that handles two routes.

  1. Allow user to input the title of a book and a summary of the book in a text area. Use POST.

  2. Use GET to display a page that looks nice and shows the title and summary back to the user (use template engines for this (Use EJS, because this is what we are going to continue to use for the rest of the semester)
