Overview

After completing the assigned reading for the week and your team activity, you are prepared to complete your personal assignment. Your assignment will consist of creating a form, sending data to the server, then displaying that data in a different part of your webpage. This will help you learn how to work with data, pages, and client-server communication.

Instructions

Your first assignment will be to create a server that handles at least two different endpoints.

  1. Build a simple form with two inputs

  2. Parses inputs in the server

  3. Displays them back to the user in a separate div

  4. Be sure to use nodemon and add npm scripts
