Overview

For the team activity this week, you will be using what you learned from the Online Videos Course to create a Node.JS server that will expose several web endpoints in your local environment (i.e., accessed through localhost:3000 and not on the web).

Bellow are optional steps to file organization that will make your life a lot easier:

  1. Create a new folder in your class GitHub repository named “teamActivities” or another name of your choosing. This is a good way to organize your work and have the code available to others on your team and serve as a possible backup.

  2. Within the “teamActivities” folder, create a new folder named “ta01” or something similar.

  3. Within the ta01 folder, create an app.js file. This is the file you will run locally with the “node” or “nodemon” commands.

Once you are ready, complete the following core challenges:

  1. Listen on port 3000 of your localhost

  2. Create web endpoint "/", which writes to an html page with <title> and <body> content of your choosing.

  3. Create web endpoint "/activities" which writes an html page which contains:

    • A list of 3+ activity names (“soccer”, “basketball”, etc) specified in hardcoded JavaScript array

    • A <form> containing a single text <input> and submit <button> which when clicked submits a post request to “/add-activity”

  4. Create a web endpoint "/add-activity" that handles the body of the POST request and console.log() the text input from the form in “/activities”, and redirect to “/activities”.

Once you have finished the core challenges above, you’ll be qualified for a 93% grade on the team activity. It’s encouraged use any extra time to experiment with your work and research/test features not covered so far. To earn 100%, it’s encouraged to complete at least one of these stretch challenges:

  1. Add some CSS for styling for a new or existing page.

  2. Write your form’s text input to a file using the ‘fs’ package.

  3. Read two number inputs from a form and outputs the sum to the console.
