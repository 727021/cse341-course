Overview

For the team activity this week, you will be using what you learned from the Online Videos Course to create a Node.JS server that will expose several web endpoints in your local environment (i.e., accessed through localhost:3000 and not on the web).

Below are steps to use a class template for file organization that will make your life a lot easier:

    Clone the class template. The template can be found here: cse341-project

    A few steps to complete the installation:

  1. Assure that you are currently in the cse341-project folder when you type in the following commands

  2. In the terminal type npm install express --save. This will install all the necessary files needed to use express.

  3. In the terminal type npm install nodemon --save. While not necessary 'nodemon' allows for your localhost server run again every time your files are saved.

  4. In order to start up a localhost session either type in nodemon if you installed nodemon or npm start if nodemon was not installed

  5. Type localhost:5000 in your browser url (This is a local workspace for you to test your code)

    6. Using the EJS templating engine is recommended, as both the class Team Activity solutions and Online Videos Course will be using it often. However, Pug and Handlebars are valid alternatives.

Once you are ready, complete the following core challenges:

  1. Create web endpoint "/", which writes to an ejs page with <title> and <body> content of your choosing.

  2. Create web endpoint "/activities" which writes an html page which contains:

    • A list of 3+ activity names (“soccer”, “basketball”, etc) specified in hardcoded JavaScript array

    • A <form> containing a single text <input> and submit <button> which when clicked submits a post request to “/add-activity”

  3. Create a web endpoint "/add-activity" that handles the body of the POST request and console.log() the text input from the form in “/activities”, and redirect to “/activities”.

Once you have finished the core challenges above, you’ll be qualified for a 93% grade on the team activity. It’s encouraged use any extra time to experiment with your work and research/test features not covered so far. To earn 100%, it’s encouraged to complete at least one of these stretch challenges:

  1. Add some CSS for styling for a new or existing page.

  2. Write your form’s text input to a file using the ‘fs’ package.

  3. Read two number inputs from a form and outputs the sum to the console.
