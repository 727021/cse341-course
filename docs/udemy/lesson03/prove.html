Assignment

  1. Organize your personal project in such a way that it follows the MVC pattern.

  2. Create JSON file that has the data you need for your app (for example, all of the products you are going to be selling).

  3. Create a webpage to get information from the json.

  4. Make sure your app has dynamic routes and paths that make sense for your app. One example is adding the specific ID of your product to your url path.
