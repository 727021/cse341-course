Project 2 Requirements

Overview

For the second half of the semester we will be building web applications using Node.js.

Basic Guidelines

One of the strengths of Node.js is its ability to expose web services which can then be consumed by client-side AJAX.

Along these lines, your project will be a full stack Node.js project that uses Web Services for the communication between the client and the server.

Guidelines

For this first project, you are to create an application of your choosing. Your project will be evaluated as to how well it:

  1. Presents a professional look and feel

  2. Accomplishes a meaningful task / does something interesting

  3. Demonstrates proficiency in Node.js and Express principles

  4. Demonstrates mastery of Client-side and Server-side communication

  5. Successfully incorporates a data layer that shows expertise in creating, reading, updating and deleting data in that database from the Node.JS application.

  6. Incorporates a login and authentication system that is meaningful that allows for constrained views based on roles after login.

  7. Demonstrates web security principles

  8. Exhibits modern and organized web architecture
