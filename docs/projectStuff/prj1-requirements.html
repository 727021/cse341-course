Project 1 Requirements

Overview

For the first half of the semester we will be building web applications using Node.js.

Basic Guidelines

One of the strengths of Node.js is its ability to expose web services which can then be consumed by client-side AJAX.

Along these lines, your project will be a full stack Node.js project that uses Web Services for the communication between the client and the server. We will not be using client-side frameworks for this project, but after completing it, you will be very well prepared to use them for future endeavors.

Guidelines

For this first project, you are to create an e-commerce application. You may choose any type of products to sell, but your project will be evaluated as to how well it:

  1. Presents a professional look and feel

  2. Accomplishes a meaningful task / does something interesting

  3. Demonstrates understanding of Node.js and Express principles

  4. Demonstrates mastery of Client-side and Server-side communication

  5. Successfully incorporates MongoDB for the data layer and shows expertise in creating, reading, updating and deleting data in that database from the Node.JS application.
